## 備註

* `了`，`瞭解`現在使用較少，統一寫作`了解`，不再對應漢字`瞭`
* `万`，統一寫作`萬`，`万俟`只在姓氏時使用
* `千`，`鞦韆`現在使用較少，統一寫作`秋千`，不再對應漢字`韆`
* `只`，`衹是`現在使用較少，統一寫作`只是`，不再對應漢字`衹`
* `冬`，擬聲`鼕`，現在使用較少，不再對應漢字`鼕`
* `饥`，漢字對應`饑|飢`，用法一致，統一寫作`饑`
* `汇`，漢字對應`匯|彙`，意思相近，統一寫作`匯`
* `夸`，統一寫作`誇`，不再對應漢字`夸`
* `合`，現在一般不用作`閤`，不再對應漢字`閤`
* `冲`, 沒有找到`冲`的用法，故統一為`衝`
